Aigeon

macOS 悬浮层，用来查看当前 Claude Code 和 Codex 任务状态。

下载 DMG 使用说明

English: index.html

安装

  1. 下载 Aigeon_0.0.2_x64.dmg
  2. 打开 DMG
  3. 将 Aigeon.app 拖到 Applications

使用

  1. 启动 Aigeon.app
  2. 在悬浮条查看当前任务状态
  3. 单击悬浮条打开任务列表
  4. 单击任务行跳回对应终端窗口
  5. 双击悬浮条切换标准模式和简单模式

首次打开被拦截

  • 当前版本没有签名，也没有 notarization
  • 如果被拦截，到 系统设置 > 隐私与安全性 里允许应用后再重新打开
