Aigeon

A macOS overlay for watching active Claude Code and Codex task status.

Download DMG Download TAR.GZ Usage Guide

Chinese: index.zh-CN.html

Install

  1. Download the DMG or TAR.GZ package
  2. If you use the DMG, open it and drag Aigeon.app into Applications
  3. If you use the TAR.GZ, extract it and move Aigeon.app into Applications

Usage

  1. Launch Aigeon.app
  2. Watch the current task status in the overlay
  3. Click the overlay to open the task list
  4. Click a task row to jump back to the matching terminal window
  5. Double-click the overlay to switch between Standard Mode and Simple Mode

First Launch Warning

  • This build is unsigned and not notarized
  • If macOS blocks it, allow the app in System Settings > Privacy & Security and relaunch it
